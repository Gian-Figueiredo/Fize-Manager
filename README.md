# Introdução
Isto é um relato sobre a produção de um gerenciador de finanças.<br>
Para quem baixar e quiser usar, primeiro inicie o ambiente virtual:
### Windows
.venv\Scritps\Activate
### Linux
source .venv\bin\Activate
